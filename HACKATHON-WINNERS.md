## Patterns: What Wins Hackathons

### Most Common Winner Types (by frequency):

1. **AI/ML + Real Data = Insight** (30% of winners)
   - Predictive models, classification, anomaly detection
   - Examples: ECOTEN heat map, BrusAir air quality, SKGRID bus optimization, GainForest deforestation

2. **Gamification + Sustainability** (20%)
   - Make eco-behavior fun, reward good choices
   - Examples: Isak (plogging game), BICIFICATION (cycling rewards), CleanWithTeam, ReMaskit, CrowdFree

3. **Citizen-as-Sensor / Crowdsourcing** (15%)
   - Leverage phones/people as data collectors
   - Examples: bAIcycle, WasteNoTime, SnapMap, UnderBranc, withthegrid

4. **Platform Connecting Supply & Demand** (15%)
   - Match people who have X with people who need X
   - Examples: Roodkappje, Telehelp, TheHeroLoop, Starvelp, OpenStore, GetWage

5. **Visualization of Hidden Data** (10%)
   - Make invisible problems visible and actionable
   - Examples: GeoFluxus waste flows, Gentrification Index, Green Land Dashboard, GreenWatch

6. **Heritage + New Tech** (10%)
   - AR/VR/3D applied to cultural sites
   - Examples: MauAR, Alt-Berlin, GaidARi, helvetiX, My Monastery, VR Kobzar

### What Judges Consistently Reward:

| Factor | Evidence |
|--------|----------|
| **Working demo** | GridGuard, bAIcycle, Isak all won with live demos |
| **One clear user + problem** | "Cyclists in Brussels" > "smart city platform" |
| **Emotional resonance** | MauAR (Berlin Wall), Marbles of Remembrance, MarjanWatch |
| **Novel data combination** | Satellite + citizen (WasteNoTime), mobility + crowd (CrowdFree) |
| **Immediately deployable** | Tranquili'score, DriveLess Routes, Smart Watering |
| **Scalable beyond one city** | GeoFluxus (11 EU regions), BICIFICATION (3 countries) |
| **Aligns with EU policy** | CROZ RenEUwable, SPIRE, Nettuniani all reference EU missions |

### Tech Stacks That Win:

| Stack | Examples |
|-------|----------|
| **Flutter + Firebase + ML** | Google Solution Challenge winners (Isak, Saheli, HeadHome) |
| **Python + Satellite + Visualization** | CASSINI winners, ECOTEN, GreenWatch |
| **React/Next.js + Maps + API** | Mobility and urban projects |
| **IoT (ESP32) + Dashboard** | DriveLess Routes, AirSplit, Smart Greenhouse |
| **LLM/GPT + Domain Data** | BlueRider, SpeechMaster, VillageTravel Bot |
| **AR.js or Unity + Heritage Data** | MauAR, GaidARi, Alt-Berlin |
